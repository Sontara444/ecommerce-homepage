# Kilangi Jewelry Homepage

A responsive, pixel-perfect eCommerce homepage for Kilangi Jewelry, featuring a modern design with mobile-first aesthetics.

## Features

### Mobile Experience
- **Mobile-First Design**: Optimized layouts for small screens.
- **Horizontal Sliders**: Smooth carousel interactions for:
  - Bestseller Products
  - Category Quick Links (Pills)
  - "Gifts That Celebrate" Section
  - Customer Reviews
- **Stacked Layouts**: Optimized "Ethos" and "Future" sections for readability on mobile.
- **Sticky Navigation**: Easy access to search and menu.

### Desktop Experience
- **Clean Layout**: Spacious grids and high-quality imagery.
- **Hover Effects**: Interactive product cards and buttons.
- **Hidden Mobile Elements**: Mobile-specific sections (like the bottom Gallery or duplicate Category links) are hidden to maintain a clean desktop look.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Layouts (Grid, Flexbox), Responsive Design (Media Queries), Custom Properties (Variables).
- **JavaScript**: (Included in `assets/index.js` for interactivity).

## Setup & Usage
1.  **Clone or Download** the repository.
2.  Open `index.html` in your web browser.
3.  Resize the browser window or use DevTools (F12) to view the **Mobile** layout (max-width: 500px).

## Project Structure
- `index.html`: Main homepage structure.
- `css/style.css`: Global styles and desktop layout.
- `css/responsive.css`: Mobile and tablet specific styles.
- `assets/images/`: Product and UI images.
- `assets/index.js`: Frontend logic.
